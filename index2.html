  • Data :- Data refers to raw facts, information, or statistics that are collected, stored, and processed by computers and other information systems.
    Data: 35, 40, 50, 60

  • Information :- Information is the processed and meaningful representation of data.
    Information: Average age of a group of people is 46.25

  • Metadata :- Metadata is data that provides information about other data. In simpler words, metadata provides essential details about data, making it easier to understand, manage, and use effectively.

  • Data Dictionary :- A data dictionary is an information repository which contains metadata.

  • Data Warehouse :- A data warehouse is an information repository which stores data.

  • Field :- A field is a character or group of characters that have a specific meaning. In simple terms, a field can be thought of as a column in a spreadsheet or a cell in a table.

  • Record/Tuple :- A record, also known as a row, is a complete set of related data within a database table.

  • Data Abstraction :- Data abstraction is like using a remote control for a TV. You don't need to know how the TV works inside; you just press buttons to change channels, adjust volume, and turn it on or off. Similarly, data abstraction lets you interact with complex things (like databases or computer programs) using simple, easy-to-understand commands or interfaces, without worrying about the intricate details of how they work behind the scenes. It simplifies things for you by hiding the complexity and showing only what you need to use.
